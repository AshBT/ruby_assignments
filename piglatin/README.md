# Pig Latin

Pig Latin is a language game of altering words. To form the Pig Latin of an English word the first consonant (or group of consonants) is moved to the end of the word and an 'ay' is affixed.

For example

```
  pig         ---> igpay
  computer    ---> omputercay
  freedom     ---> eedomfray
  string      ---> ingstray
```
